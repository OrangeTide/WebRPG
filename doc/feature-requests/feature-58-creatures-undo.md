# Feature 58: Creatures Undo Button

Undo button on the Creatures table to reverse the last action.

## Description

Add an undo button on the Creatures table to undo the last action. Supported actions to undo:

- Delete
- Field change
- Rename
- Place on map
- Roll initiative
