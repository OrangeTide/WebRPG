# Feature 6: Progress clocks

Make a progress clock window. These are persistent to the game session, and there can be multiple of them with their own setings and state. A progress clock is a circle that is divided up into a number of pie-shaped wedges that are either filled or unfilled. The number of wedges is fixed when it is created. The filled and unfilled colors are configurable for each instance. The progress clock always fills from the top most and in a clockwise order. There are two buttons on the window beneath the progress clock, add (+) and remove (-). These increment or decrement the progress clock, causing another wedge to become filled or unfilled respectively.
each window instance can be named with a label and have a small description area beneath it (editable).
a lock button in the upper right corner prevents increment/decrement or editing of the description or labels.
