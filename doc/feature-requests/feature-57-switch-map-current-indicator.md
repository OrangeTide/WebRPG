# Feature 57: Switch Map Current Map Indicator

Show the current map in the switch map dropdown, grayed out and not selectable.

## Description

The switch map pull-down should include the current map on the list, but have it grayed out and not selectable. This is a convenience UI feature so the user can see which map is active.
