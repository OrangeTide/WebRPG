# Feature 65: Character Sheet Edit Lock

Lock button on character sheets to control who can edit.

## Description

Add a lock button on a toolbar area at the top of each character sheet. Clicking it cycles through three permission states:

- **Open** — allows edits from all players/GMs clients
- **Owner only** — allows only the controlling player(s) to edit
- **GM only** — allows edits only from GM clients; only selectable by GM clients, grayed out for non-GM

The icon appearance changes to reflect the current state. When the current client does not have permission to edit, the lock button and all items on the popup are disabled and grayed out.
