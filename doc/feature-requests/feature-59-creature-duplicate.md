# Feature 59: Creature Duplicate Button

Duplicate button on the creature menu to clone a creature as a template for a new type.

## Description

Add a duplicate button on the creature menu so that the creature's attributes, name, icon, etc. can be used as a template for a new creature type. The duplicate creates a new creature entry with the same fields, allowing the user to modify it independently.
